# 🧭 TourMate — The Perfect AI-Generated Travel Itinerary Planner

**TourMate** is an intelligent travel itinerary generator that creates personalized, optimized, and real-time adaptable travel plans using AI and data from real-world APIs.

---

## 🚀 Features

- 🌐 **Destination-aware**: Location-specific suggestions via Google Places API
- 🧠 **AI-Personalized**: Custom itineraries based on user interests, budget, and travel style
- 📅 **Time-optimized**: Plans your day smartly based on opening hours, travel time, and pace
- 🌦️ **Real-time Aware**: Optional weather integration for adjusting your plan
- 📝 **GenAI Descriptions**: GPT-powered summaries and suggestions
- 🗺️ **Map + PDF Export**: Interactive maps and offline itinerary support
